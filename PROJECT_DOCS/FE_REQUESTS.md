# FE -> BE Requests

> FE Agent ghi khi cần endpoint chưa có. BE Agent đọc và tạo task.

---

## Format

```
[Date] Need: [Mô tả]
Method: [GET/POST/PUT/DELETE]
Path: /api/v1/...
Request: {field: type}
Response: {field: type}
Priority: HIGH/MED/LOW
Status: PENDING / DONE
```
